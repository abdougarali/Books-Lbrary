# Books Demo Page - قل ربي زدني علما

صفحة Demo احترافية لمكتبة إسلامية أونلاين بتصميم عصري وألوان Coffee Cream.

## المميزات

- ✅ تصميم احترافي عصري
- ✅ عرض الكتب في Swiper Carousels
- ✅ طلب مباشر عبر واتساب
- ✅ تصميم Responsive كامل
- ✅ Coffee Cream Color Palette
- ✅ بدون Backend
- ✅ SEO Optimized

## التقنيات المستخدمة

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Swiper.js

## التشغيل

```bash
# تثبيت الحزم
npm install

# تشغيل المشروع في وضع التطوير (على المنفذ 3004)
npm run dev
# سيعمل على: http://localhost:3004

# بناء المشروع للإنتاج
npm run build

# تشغيل النسخة المبنية (على المنفذ 3004)
npm start
# سيعمل على: http://localhost:3004
```

## التخصيص

### تغيير بيانات الكتب

عدّل ملف `data/books.ts` لتغيير الكتب المعروضة.

### تغيير رقم واتساب

في ملف `app/page.tsx`، غيّر الرقم `1234567890` في دالة `handleWhatsAppOrder` برقم الواتساب الخاص بك.

### التصميم

التصميم يستخدم Tailwind CSS. يمكنك تعديل الألوان والتصميم من خلال تعديل className في الملفات.
